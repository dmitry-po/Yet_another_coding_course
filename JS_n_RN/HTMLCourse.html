CSS

Flexbox

CSS Flexbox - технология для создания макетов с помощью инструментов выравнивания. Для того, чтобы использовать Flexbox, необходимо установить атрибут стиля display: flex. Для сравнения:

  • display: block
  • display: flex

    • Определение осей (flex-direction)

    У каждого контейнера есть 2 оси: основная и перпендикулярная. Вдоль этих осей и происходит выравнивание элементов. По умолчанию, основной осью считается горизонтальная, но значение можно переопределить с помощью атрибута flex-direction:

  • flex-direction: row
  • flex-direction: row-reverse
  • flex-direction: column
  • flex-direction: column-reverse

    • Выравнивание по главной оси (justify-content)

    Выравнивание элементов вдоль главной оси выполняется с помощью атрибута justify-content. По умолчанию устанавливается значение flex-start, но также используются:

  • justify-content: flex-start
  • justify-content: center
  • justify-content: flex-end
  • justify-content: space-around
  • justify-content: space-evenly
  • justify-content: space-between

    • Выравнивание по перпендикулярной оси (align-items)

    align-items работает аналогично justify-content, но с перпендикулярной осью. По умолчанию устанавливается значение stretch, но для его корректной работы требуется, чтобы у дочерних элементов не было задано значение параметра высоты/ширины, в зависимости от выбранной главной оси. Также применяются следующий значения:

  • align-items: flex-start
  • align-items: center
  • align-items: flex-end
  • align-items: stretch

    • Теги justify-content и align-items можно совмещать в одной верстке (как на примере выше).

    Выравнивание отдельных элементов (align-self)

    Можно выравнивать не только все элементы, но и отдельные его элементы с помощью атрибута align-self:

    align-self: flex-start
    align-self: stretch
    align-self: flex-end
    Атрибут не задан

    Размеры элементов

    flex-basis - позволяет задать абсолютное значение длины/ширины (в зависимости от основной оси) элемента контейнера:

    10%
    20%
    30%
    40%

    flex-grow - позволяет задать относительное значение длины/ширины (в зависимости от основной оси) элемента контейнера. В примере ниже первый объект занимает 1/(1+2+3+4) часть пространства, второй 2/10 и т.д.. Используется, когда необходимо, чтобы дочерние элементы занимали всю площадь контейнера:

  • Без flex-grow:
    • 30px
    40px
    50px
    60px
  • С flex-grow:
    • 30px, 1
    40px, 2
    50px 3
    60px ,4

    flex-shrink - противоположность flex-grow, позволяет "ужимать" элементы при превышении доступной ширины контейнера:

  • Все элемент имеют ширину 140px, контейнер - 600px:
    • 1
    2
    3
    4
  • Условия те же, но ширина контейнера уменьшилась до 300px:
    • 1
    2
    3
    4

    Перенос элементов (flex-wrap)

    Проблему с уменьшением доступных размеров родительского контейнера можно решить другим способом - переносом дочерних элементов на новую строку/столбце (в зависимости от основной оси) - с помощью атрибута flex-wrap (по умолчанию принимает значения nowrap, также доступны wrap и wrap-reverse):

    1
    2
    3
    4

    Межстрочное выравнивание (align-content)

    В случаях, когда в контейнере появляется несколько строк/столбцов (в зависимости от основной оси), появляется возможность использовать атрибут align-content. В отличие от align-items, который определяет выравнивание элементов внутри одной строки/столбца, align-content определяет выравнивание строк/столбцов внутри контейнера. Например:

  • align-items: center
  • align-content: center

    • Помимо значения center, атрибут принимает также значения flex-start, flex-end, space-around, space-between, space-evenly, stretch (по умолчанию).
